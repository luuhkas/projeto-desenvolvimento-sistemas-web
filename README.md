Repositório acadêmico da disciplina **Projeto e Desenvolvimento de Sistemas Web**, criado para centralizar exercícios, práticas e projetos desenvolvidos ao longo da matéria.

## Contexto

Este repositório funciona como base única para armazenar a evolução das atividades da disciplina. A proposta é manter o conteúdo organizado em um só lugar, facilitando versionamento, consulta e continuidade do trabalho conforme novas demandas forem sendo definidas.

## Objetivo

- concentrar os materiais da disciplina em um único repositório;
- registrar a evolução prática em desenvolvimento web;
- manter histórico e organização das entregas;
- servir como apoio para estudo, revisão e acompanhamento do semestre.

## Stack

As atividades deste repositório podem envolver HTML5, CSS3, Tailwind CSS, JavaScript, React e versionamento com Git/GitHub, conforme a necessidade de cada prática desenvolvida na disciplina.

## Como visualizar os projetos

As tarefas estáticas podem ser abertas diretamente no navegador.

Exemplo no macOS:

```bash
open "Tarefa 3/index.html"
```

Para testar a tarefa 4:

```bash
open "Tarefa 4/index.html"
```

Para testar a tarefa 6, instale as dependências e rode o servidor React:

```bash
cd "Tarefa 6"
npm install
npm run dev
```

Depois, abra o endereço exibido no terminal. Normalmente será `http://127.0.0.1:5173/`, mas pode mudar para outra porta se a primeira estiver em uso. No macOS, também é possível iniciar a tarefa com dois cliques em `Tarefa 6/abrir_tarefa_6.command`.
